This code based on https://github.com/laidlaw42/ableton-live-midi-remote-scripts repository.
Copy this folder to "/Applications/Ableton Live 12 Suite.app/Contents/App-Resources/MIDI Remote Scripts/TrippyFader" and restart Ableton.
Connect the device to PC.
In Ableton go to settings -> 'Link, Tempo & MIDI' and in Control Surface find TrippyFader and select it in Input and Output as well. Also select it in Input and Output ports.
You can remap buttons or fader as you want in MIDI_Map.py file.
